# fe-workflow-with-gulp

## 功能
**1. 修改文件自动刷新浏览器**

**2. scss编译压缩并自动补全前缀**

**3. js压缩**

**4. 图片压缩**

## 用法
### 安装依赖包
```
npm i
```
### 开发模式
```
npm run dev
```
开发预览：`http://localhost:8888/views/testgulp/index.html`

### 生产模式
```
npm run prod
```
生产预览：`http://localhost:8888/dest/testgulp/index.html`
